# AWS region & AZ

- AWS region: 以地理位置做區分，每一個 region 都由三個以上的 AZ 所組成。 region 和 region 間是互相獨立的，無法查詢跨區資料。
- AZ(Availability Zones): 獨立的資料中心，擁有一個以上的備用電源， AZ 間的交流是藉由光纖，有高頻寬、低延遲等特性，且所有的交流都會將資料加密。
